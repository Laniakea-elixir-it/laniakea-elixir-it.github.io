Dear Users,

the package CentOS ca-certificates package needs to be manually updated in order to restore data upload from HTTPS sources. Currently, the upload module will display the following error if data are uploaded through HTTPS link: Unable to fetch https://data-url To update the package login on your instance through SSH and type: sudo yum update -y ca-certificates
